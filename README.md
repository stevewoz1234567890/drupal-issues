## Example snippets

Illustrative only‚ÄĒadapt names, versions, and Selenium endpoints to your project. For **Chrome** / **`/dev/shm`** on hosted runners, see [`community/gha-ubuntu-chrome-behat-shmem-psa.md`](community/gha-ubuntu-chrome-behat-shmem-psa.md).

### GitHub Actions ‚ÄĒ pin Ubuntu 22.04 on the job

Avoids the **Ubuntu 24.04** / **linux-azure-6.17** path some pipelines hit with headless Chrome; use when a runner-image bump breaks browser tests and you need a quick isolation step.

```yaml
jobs:
  phpunit:
    runs-on: ubuntu-22.04
    steps:
      - uses: actions/checkout@v4
```

### Behat (Mink) ‚ÄĒ Chrome flag `--disable-dev-shm-usage`

Makes Chrome use **`/tmp`** instead of **`/dev/shm`**, which sidesteps shared-memory issues on CI. Your `behat.yml` structure may differ; this matches common **MinkExtension** + **selenium2** setups.

```yaml
default:
  extensions:
    Behat\MinkExtension:
      browser_name: chrome
      selenium2:
        wd_host: 'http://127.0.0.1:4444/wd/hub'
        capabilities:
          browserName: chrome
          chromeOptions:
            args:
              - '--disable-dev-shm-usage'
```

### Drupal module ‚ÄĒ `core_version_requirement`

Example of declaring **Drupal 10** and **11** in a single extension (adjust lower bound to match APIs you use):

```yaml
name: 'Example module'
type: module
description: 'Illustrative core_version_requirement only.'
package: Custom
core_version_requirement: ^10.3 || ^11
```
